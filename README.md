# zap-link
criador simples de api para zapzap (link para chats) @bydeathlxncer

instalação:

pkg install git

git clone https://github.com/bydeathlxncer/zap-link

cd zap-link

chmod +x *

bash zap-link.sh

agora insira o número e um texto para chat!
